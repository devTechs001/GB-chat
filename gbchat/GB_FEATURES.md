## 🎯 OVERVIEW

GBChat now includes **50+ GB WhatsApp features** that enhance privacy, customization, and messaging capabilities. All features are accessible through the **GB Features** section in Settings.

### Quick Access
- **Settings → GB Features**: Access all GB features
- **Floating Action Button**: Quick access to starred, pinned, and scheduled messages
- **Message Context Menu**: Pin, star, or schedule messages directly from chat

## 🔒 PRIVACY FEATURES

### 1. Hide Online Status
**What it does**: Appear offline while still seeing others' online status  
**How to enable**: Settings → GB Features → Privacy → Hide Online Status  
**Use case**: Browse without others knowing you're active

### 2. Freeze Last Seen
**What it does**: Keep your last seen timestamp frozen at a specific time  
**How to enable**: Settings → GB Features → Privacy → Freeze Last Seen  
**Use case**: Maintain privacy while still using the app

### 3. Hide Blue Ticks
**What it does**: Read messages without sending read receipts  
**How to enable**: Settings → GB Features → Privacy → Hide Blue Ticks  
**Use case**: Read messages privately

### 4. Hide Second Tick
**What it does**: Hide message delivery ticks (single tick only)  
**How to enable**: Settings → GB Features → Privacy → Hide Second Tick  
**Use case**: Send messages without delivery confirmation

### 5. Hide Forward Label
**What it does**: Forward messages without "Forwarded" label  
**How to enable**: Settings → GB Features → Privacy → Hide Forward Label  
**Use case**: Share forwarded messages discreetly

### 6. Anti-Status View
**What it does**: View status updates without sender knowing  
**How to enable**: Settings → GB Features → Privacy → Anti-Status View  
**Use case**: Watch status updates anonymously

### 7. Anti-Delete Status
**What it does**: Save status updates before they get deleted  
**How to enable**: Settings → GB Features → Privacy → Anti-Delete Status  
**Use case**: Never miss a status update

### 8. Anti-Revoke
**What it does**: Prevent message deletion by sender  
**How to enable**: Settings → GB Features → Privacy → Anti-Revoke  
**Use case**: Keep messages even if sender deletes them

### 9. View Once Bypass
**What it does**: Save view-once media permanently  
**How to enable**: Settings → GB Features → Privacy → View Once Bypass  
**Use case**: Save view-once photos and videos

### 10. Incognito Mode
**What it does**: Browse without leaving any traces  
**How to enable**: Settings → GB Features → Privacy → Incognito Mode  
**Use case**: Complete privacy while using the app

## 💬 MESSAGING FEATURES

### 1. Schedule Messages
**What it does**: Send messages at scheduled times  
**How to use**:
1. Long-press on a message
2. Click "Schedule" icon
3. Set date and time
4. Message will be sent automatically

**Quick access**: FAB → Scheduled Messages

### 2. Auto-Delete Messages
**What it does**: Automatically delete messages after timer  
**Timer options**:
- 24 hours
- 2 days
- 7 days
- 30 days

**How to enable**: Settings → GB Features → Messages → Auto-Delete Messages

### 3. Pin Messages
**What it does**: Pin important messages to top  
**How to use**:
1. Long-press on message
2. Click "Pin" icon
3. Access from FAB → Pinned Messages

**Limit**: Up to 10 pinned messages

### 4. Star Messages
**What it does**: Star messages for quick access  
**How to use**:
1. Long-press on message
2. Click "Star" icon
3. Access from FAB → Starred Messages

**Limit**: Unlimited starred messages

### 5. Chat Filters
**What it does**: Organize chats with custom filters  
**How to create**:
1. Settings → GB Features → Messages → Chat Filters
2. Create new filter
3. Name filter (e.g., "Work", "Family")
4. Select chats to include
5. Choose color

**Use case**: Quickly filter chats by category

### 6. Do Not Disturb (DND)
**What it does**: Silence notifications during scheduled time  
**How to set**:
1. Settings → GB Features → Messages → Do Not Disturb
2. Set start and end time
3. Enable exceptions for important contacts

**Default**: 10:00 PM - 7:00 AM

## 🔌 API REFERENCE

### Base URL
```
/api/gb-features
```

### Authentication
All endpoints require Bearer token authentication:
```
Authorization: Bearer <token>
```

### Endpoints

#### Get GB Features
```http
GET /api/gb-features
```

#### Update GB Features
```http
PUT /api/gb-features
Body: {
  "section": "privacy",
  "data": { "hideBlueTicks": true }
}
```

#### Toggle Feature
```http
POST /api/gb-features/toggle
Body: {
  "section": "privacy",
  "feature": "hideBlueTicks"
}
```

#### Pin Message
```http
POST /api/gb-features/pin
Body: {
  "messageId": "xxx",
  "chatId": "yyy"
}
```

#### Unpin Message
```http
POST /api/gb-features/unpin
Body: {
  "messageId": "xxx"
}
```

#### Get Pinned Messages
```http
GET /api/gb-features/pinned
```

#### Star Message
```http
POST /api/gb-features/star
Body: {
  "messageId": "xxx",
  "chatId": "yyy"
}
```

#### Unstar Message
```http
POST /api/gb-features/unstar
Body: {
  "messageId": "xxx"
}
```

#### Get Starred Messages
```http
GET /api/gb-features/starred
```

#### Schedule Message

## 📖 USAGE GUIDE

1. **Access GB Features**
   - Open Settings
   - Click on "GB Features" tab
   - Explore all feature categories

2. **Quick Actions**
   - Click the Floating Action Button (FAB)

3. **Message Actions**

## 🐛 TROUBLESHOOTING

### Common Issues

#### Feature Not Working
**Solution**: 
1. Refresh the page
2. Clear browser cache
3. Check internet connection
4. Verify authentication token

#### Messages Not Pinning/Starring
**Solution**:
1. Check if already pinned/starred
2. Verify message exists
3. Check API response in console

#### Scheduled Messages Not Sending
**Solution**:
1. Ensure browser/app is open at scheduled time
2. Check timezone settings
3. Verify scheduled time is in future

#### Themes Not Applying
**Solution**:
1. Refresh the page
2. Clear theme cache
3. Re-select theme

#### Settings Not Saving
**Solution**:
1. Check internet connection
2. Verify authentication
3. Check browser console for errors

### Error Codes

| Code | Meaning | Solution |
|------|---------|----------|
| 401 | Unauthorized | Re-login to refresh token |
| 403 | Forbidden | Check permissions |
| 404 | Not Found | Verify message/chat ID |
| 500 | Server Error | Contact support |
