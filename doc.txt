1. Project Overview
GBChat is a feature-rich real-time messaging platform inspired by GB WhatsApp, built with MERN stack, featuring themes, scheduled messages, message reactions, stories, channels, self-destructing messages, chat locks, and more.

Tech Stack
Layer	Technology
Frontend	React 18 + Vite + JSX
Styling	Tailwind CSS v3.4
State	Zustand + React Query
Realtime	Socket.IO Client
Backend	Node.js + Express.js
Database	MongoDB + Mongoose
Realtime Server	Socket.IO
Auth	JWT + bcrypt
File Upload	Multer + Cloudinary
Notifications	Web Push API
